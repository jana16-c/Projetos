Utilização do processador

Esta aba explica o fluxo de uso e a função de cada botão.

Escolher PDF

Seleciona o arquivo PDF que será lido. O processador trabalha melhor com PDFs textuais, ou seja, arquivos em que o texto pode ser selecionado com o mouse.

Páginas para converter

Define quais páginas serão processadas. Aceita páginas isoladas e intervalos, como 1-3, 5, 8-10. Se o campo ficar vazio, todas as páginas serão convertidas.

Modo estrutural inteligente

Usa a posição real dos textos no PDF para reconstruir linhas e colunas. É o modo recomendado para tabelas, demonstrativos, guias e relatórios.

Grade visual por posição

Preserva melhor o alinhamento visual quando o PDF não possui colunas constantes. Pode gerar mais colunas vazias, mas ajuda em layouts irregulares.

Tolerância de linha

Controla o agrupamento vertical dos textos. Aumente se uma mesma linha estiver sendo quebrada em duas. Diminua se linhas diferentes estiverem sendo juntadas.

Tolerância de coluna

Controla o agrupamento horizontal das colunas. Aumente se valores da mesma coluna estiverem caindo em colunas diferentes. Diminua se colunas vizinhas estiverem sendo misturadas.

Separação de célula

Define quando dois blocos de texto próximos devem continuar na mesma célula ou virar células separadas. É útil em PDFs que quebram palavras e valores em muitos fragmentos.

Uma aba por página

Quando marcado, cada página convertida vira uma aba separada no Excel. Quando desmarcado, todas as páginas ficam em uma única aba.

Mesclar títulos detectados

Quando marcado, linhas reconhecidas como títulos são mescladas no Excel para preservar a leitura do relatório.

Processar páginas

Lê as páginas selecionadas, reconstrói a estrutura e gera uma prévia. Os botões de exportação só são liberados após esse processamento.

Exportar Excel

Gera um arquivo .xlsx com as tabelas extraídas, formatação funcional, títulos e uma aba de diagnóstico.

Exportar ZIP completo

Gera um pacote com Excel, CSV por página e JSON técnico da extração. Use este modo quando precisar conferir ou auditar o resultado.

Precisão

Como a extração funciona

Este conversor trabalha com PDFs textuais. Em PDFs escaneados, a camada de texto não existe e será necessário OCR. A extração é baseada nas coordenadas reais do PDF, não em simples cópia de texto.

Linhas: os fragmentos são agrupados pela posição vertical e pelo tamanho da fonte.
Colunas: o processador procura alinhamentos horizontais repetidos ao longo da página.
Células: textos próximos são unidos; lacunas maiores indicam separação de célula.
Diagnóstico: o Excel inclui uma aba com quantidade de linhas, colunas, itens de texto, confiança e avisos.
